# avoiding-football-lottery-pitfalls

## Purpose

Use this skill before finalizing any football lottery analysis or ticket structure.

It checks for common reasoning mistakes.

## Pitfalls

Reject or warn if the analysis contains:

1. Low SP equals safe.
2. SP drop equals certain win.
3. High SP equals valuable upset.
4. Ignoring hhad confirmation.
5. Ignoring ttg conflict.
6. Treating A priority as win probability.
7. Calling structure confidence true probability.
8. Using overseas bookmaker logic for Sporttery SP.
9. Recommending same-match multiple play types in illegal parlay.
10. Ignoring 90-minute result rule.
11. Ignoring draw risk in win/lose double selection.
12. Overreacting to one low-quality public source.
13. Making a deterministic prediction from public news.
14. Treating missing data as negative evidence.
15. Ignoring late-window reversal.
16. Overtrusting high-volatility signals.
17. Using public news to replace SP structure.

## Forbidden Words

If final output contains any of these, revise before output:

- 必胜
- 稳胆
- 稳赚
- 包中
- 稳赢
- 真实胜率
- 模型预测概率
- 建议下注
- 下注金额
- 梭哈

Allowed alternatives:

- 结构较清晰
- 值得人工继续研究
- 方向增强
- 风险较低
- 不适合作为核心方向
- 市场表达更连贯
- 结构置信度，不是赛果概率

## Required Final Check

Before output, confirm:

- Is this framed as market expression?
- Are risks included?
- Are conflicts included?
- Is missing data handled correctly?
- Is late-window reversal handled?
- Is high volatility handled?
- Is the recommendation limited to human research focus?
- Is banned language absent?
